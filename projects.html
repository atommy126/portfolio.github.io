WARD

Personal Project
WARD is a 2D dungeon-crawler roguelike game currently under development. Your goal is to explore an overrun research facility to close the rift that lays at bottom, all while attempting to survive the anomalies that wait within.

As this is an independent project, I have been handling every aspect of the development on my own, including UI/UX design, gameplay programming, creation of art assets, and much more.

Crazy Crops

Academic Project
Crazy Crops is a tabletop farming card game developed by a group of students. Cast spells, plant seeds, and harvest crops to weed out the competition and control the most prosperous farm in the world!

I designed the game's mechanics, rules, and card layouts. I oversaw playtests and analyzed data to balance the game and adjust the rules. I also oversaw the design documentation to make sure the team was keeping it accurately up to date.

Audio Visualizer

Academic Project
The audio visualizer is a webpage that creates visuals using audio waveform data. It includes controls to turn on and off different visual elements, as well as controls for manipulating the waveform data to change the audio.

This project was done largely in JavaScript, which included the use of the Canvas and Web Audio APIs. Styling was completed using CSS.

Auxiliary Power

Academic Project
Auxiliary Power is an interactive story set in the laboratory of WARD. You find your suit critically damaged, running only on auxiliary systems. With no auditory or camera systems left to help you navigate the facility, you must escape using descriptions from the suit’s autonomous co-pilot.

I created the story, map, and visuals for this short experience. The project was created in Twine and styled using CSS.

Forlorn Maze

Academic Project
Forlorn Maze is a tile-based puzzle game. You must navigate through an ancient maze to find the treasure at the end. However, strange magic and traps make navigating this labyrinth more difficult than any ordinary maze.

Forlorn Maze was made using PixiJS, and was based on an old Python game I created. This was an independent project based, so I handled all of the programming, artwork, and level design.

Descent

Academic Project
Descent is a scene set in the world of WARD. You are at the top of the facility, preparing to take the elevator down into the heart of the anomalous zone.

This project was created using Unity. I created all of the models using Maya, and textured them using Substance Painter. I then assembled the scene in Unity and made use of lighting settings to create a dark, foreboding mood within the scene.

 

Craft-Up

Academic Project
Craft-Up is a top-down survival-craft game. You must explore the world to find resources that you can use to create tools and weapons. Monsters emerge at nightfall to take your life, so you must be prepared to fight back. As more powerful enemies appear, you gain access to their materials, allowing you to craft powerful weapons and become unstoppable.

I worked with a team of other students to create this game. I handled the programming for collisions and was in charge of enemy behavior. I made an external tool to streamline enemy creation, which would output enemy files that the game would read from.
